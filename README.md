# Assignment 2 for CS F364 Design And Analysis of Algorithms

## Guides

To build the project, execute -

```
$ make all
```

To run the project, execute -

```
$ build/algo
```

To run the tests, execute - 

```
$ make test
```

To delete all the build files, execute -

```
$ make clean
```
